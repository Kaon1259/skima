⚡ 단바 — 오렌지 + 번개 통합 6종

번개를 더 적극적으로 섞은 디자인. F12 → 요소 선택 → "Capture node screenshot" 으로 PNG 추출.

1 · 큰 번개 배경 (가장 직관적)

번개를 배경에 크게, 단바 텍스트 오버레이

단바
번개가 부드럽게 배경 — 워크마크 스타일

2 · 번개를 단바 옆에 (병렬)

번개 = 글자 한 자처럼 동등 비중

단바
⚡ 단바 — 균형 있는 워드마크

3 · 사선 번개 컷 (다이내믹)

번개 모양으로 두 색 분할, 글자는 위에 오버레이

단바
스피드 강조, 그래픽적 임팩트

4 · 노란 번개 + 흰 단바 (대비 강함)

번개는 노랑(에너지), 단바는 흰색

단바
색 대비로 번개·이름 모두 또렷

5 · 원형 + 우상단 번개

iOS 앱 아이콘 친화 — 원형 자체가 액센트

단바
베이지 배경 + 오렌지 원 + 노란 번개

6 · 단 글자 위에 번개 오버레이 (실험적)

번개가 글자 위에 살짝 겹침 — 입체감

단바
번개가 글자 사이를 가로지름

💡 추출: Chrome F12 → 화살표 아이콘으로 .icon-frame 선택 → 우클릭 → "Capture node screenshot" → PNG 자동 다운로드 (정사각).
512×512 필요시: CSS의 .icon-frame { width: 512px; height: 512px; } 로 변경 후 폰트도 ×2 (예: 110px → 220px), 캡처.
