## Part 5. 페르소나 심화 — 누구한테 팔고 누가 일할 것인가

### 5-1. 워커 페르소나 (공급측)

일본 Timee 실제 사용자 분포는 **30~50대가 70%**, 학생은 15% 수준. 통념과 다름. 한국 적용 시 우선순위 재배치 필요.

| 페르소나 | 비중 가설 | 주 시프트대 | 후크 | 마케팅 채널 |
|---|---|---|---|---|
| **2030 N잡 직장인** | 35% | 평일 저녁/주말 | 부수입, 본업 외 유연성 | 인스타·잡플래닛·블라인드 광고 |
| **Z세대 학생 (대학·고졸)** | 25% | 평일 오후/주말 | 즉시 입금, 면접 부담 X | 에브리타임·인스타·틱톡 |
| **4050 경력단절 여성** | 25% | 새벽/오전 | 짧은 시간, 베이커리 친숙도 | 맘카페·당근·카카오 채널 |
| **5060 시니어** | 15% | 평일 오전/오후 | 단순 업무, 사회 참여 | 시니어플러스·복지관 |

**시사점**:
- 처음부터 학생만 노리면 안 됨 — N잡 직장인 풀이 가장 안정적
- 4050 여성을 베이커리 새벽 시프트로 매칭 = 차별적 무기
- 시니어 풀은 5~7년 후 본격화, 미리 인프라 깔아둘 가치 있음

### 5-2. 점주 페르소나 (수요측)

저가 프렌차이즈 가맹점주는 **평균 1.2~1.5개 매장 운영**, 본인 상주율 50~60%. 의사결정 패턴이 일반 자영업자와 다름.

| 점주 유형 | 비중 | 의사결정 핵심 | 영업 접근법 |
|---|---|---|---|
| **본사 충성 가맹점주** | 50% | 본사 권장 = 사실상 채택 | **본사 딜이 절대적** |
| **자율형 가맹점주** | 25% | 주변 점주 입소문 + 본인 ROI 계산 | 지역 거점 점주 1명을 챔피언으로 |
| **개인 카페 사장** | 20% | 가격 + 즉시성 | SNS·당근 광고, 무료 체험 |
| **베이커리 가맹점주** | 5% | 새벽 인력 = 절대 페인 | 새벽 시프트 보장 약속 |

**핵심 인사이트**:
- 가맹점주의 75%는 본사 또는 주변 점주 영향 → **1:1 영업은 비효율**, 거점 만들기가 답
- 디지털 리터러시는 카카오톡·배민·요기요 수준 — **새 앱은 진입 저항**, 카카오톡 알림톡 + 웹 대시보드 우선이 좋음

## Part 6. 단위 경제학 — 숫자로 검증

### 6-1. 시프트 1건 기준

| 항목 | 금액 | 비고 |
|---|---|---|
| 평균 시프트 | 4시간 × 10,030원 | = **40,120원** 임금 |
| Skima 수수료 (12% 가정) | 4,814원 | 점주 부담 |
| 점주 총 부담 | **44,934원** | 임금 + 수수료 |
| 워커 수령 | **40,120원** | 일급 15만원 미만 → 원천징수 0 |

→ 워커가 거의 풀 시급 수령. **즉시 입금 후크가 강하게 작동**.

### 6-2. 매장당 / 워커당 LTV

**매장 가정**:
- 월 평균 알바 시프트 70건 중 Skima 사용 20% = 14 시프트/월
- 매장당 Skima 매출: 14 × 4,814원 = **67,396원/월**
- 점주 평균 활성 기간 12개월 → 매장 LTV = **약 81만원**

**워커 가정**:
- 활성 기간 6개월, 월 4 시프트
- 워커가 발생시키는 GMV: 24 × 40,120원 = 96만원
- 그 중 수수료(12%) = **11.5만원/워커 LTV**

**CAC**:
- 워커 1인: 광고 + 추천 인센티브 ≈ **1만원**
- 점주 1매장 (직접 영업): 약 **7.5만원**
- 점주 1매장 (본사 딜): **거의 0원**

**LTV/CAC**:
- 워커: 11.5x ✅
- 점주 (직접): 10.8x ✅
- 점주 (본사 딜): >100x ✅✅✅ — **본사 딜의 압도적 우위가 숫자로 증명**

### 6-3. 시나리오별 ARR

| 시나리오 | 매장 수 | ARR | 시점 |
|---|---|---|---|
| 강남 1개 동 파일럿 | 100 | **약 8,000만원** | 6개월차 |
| 서울 5개 동 확장 | 500 | **약 4억원** | 18개월차 |
| **본사 1곳 딜 (컴포즈/더벤티 가맹점 일부)** | 1,000 | **약 8억원** | 24개월차 |
| **본사 1곳 전면 딜** | 2,500 | **약 20억원** | 36개월차 |
| 본사 2~3곳 딜 | 6,000 | **약 50억원** | 48개월차 |

→ **파일럿만으로는 사업 안 됨, 본사 딜이 결정적**. 24개월 안에 본사 1곳 딜 못 잡으면 사업 불성립.

## Part 7. 노쇼 백업 SLA — 정량 모델링

"1시간 내 백업 매칭 80%" 약속을 지키려면 워커 풀이 얼마나 필요한가?

### 가정
- 강남 1개 동 카페 100개
- 매장당 일 평균 0.5건 시프트 발생 → **일 50건 시프트**
- 일본 Timee 평균: 시프트 1건당 응답 3~5명 → 보수적으로 **4명**
- 활성 워커가 일 평균 0.4회 시프트 응답 (실제 0.3~0.5)

### 계산
- 필요 응답 수: 50건 × 4명 = **일 200 응답**
- 활성 워커 1인 일 0.4 응답 → 활성 워커 **500명** 필요
- 등록 워커 / 활성 워커 비율 일반적으로 4~5x → **등록 워커 2,000~2,500명** 필요

### 일본 Timee 벤치 비교
- Timee: 등록 워커 900만 / 등록 사업장 35만 = **25:1 비율**
- 우리 모델: 2,500명 / 100매장 = **25:1** — 비율 일치 ✅

**시사점**: 강남 파일럿 워커 동원 목표 = **2,500명 등록 / 500명 활성**. 이게 "1시간 매칭 80%"의 정량 조건. 워커 동원이 1순위 운영 KPI.

## Part 8. 법적 함정 — 한국 실제 판례

한국에서 **사업소득자(3.3%) 모델**이 절대 안 되는 이유 — 실제 판례·사례.

### 8-1. 배민커넥트 근로자성 인정 (2022~2024)
- 일부 라이더에 대해 근로자성 인정 판결
- "지휘·종속성"이 강하면 사업자 계약이어도 근로자로 재분류
- 카페 알바는 라이더보다 종속성 더 강함 (시간·장소·업무 모두 점주 통제)
- → **사업소득자 모델 시 100% 위장도급 적발 위험**

### 8-2. 쿠팡플렉스 / 요기요 라이더 산재 (2020~)
- 특수형태근로종사자 산재 의무 가입 확대
- 노조 결성 → 단체교섭 압박
- → 플랫폼이 "고용주가 아니다"라고 주장해도 사회적·법적 압박 받음

### 8-3. 타다 사태 후속 입법 (2024)
- 플랫폼노동자 보호법 진행 중
- "플랫폼이 일감 분배·평가·가격 결정하면 사용자성 인정" 방향
- → Skima Byte가 시급·시간 강제 X, 점주가 결정하는 구조여야 함

### 8-4. 안전한 모델 — 정리

| 요소 | 권장 설계 |
|---|---|
| 고용주 | **점주가 일용근로자로 직접 고용** |
| 플랫폼 역할 | 매칭·정산·평가 인프라 (사용자 X) |
| 시급/시간 결정 | **점주가 결정** (플랫폼은 추천만) |
| 산재 가입 | 일용근로자 산재 의무 → 점주 명의로 자동 신고 |
| 원천징수 | 일용근로 6.6% 자동 (15만원 미만 비과세) |
| 4대보험 | 같은 매장 월 8일 초과 시 자동 알림 |

→ **모든 컴플라이언스가 자동화된 SaaS = 진짜 해자**. 누구도 한국에서 이 인프라 못 깔았음.

## Part 9. 본사 영업 — 컴포즈 vs 더벤티 비교

24개월 안에 본사 딜이 사업 성립의 절대 조건. 어디부터 갈 것인가.

### 9-1. 컴포즈커피
- 매장 수: 2,600+
- 2024 졸리비푸드(필리핀) 인수, 4.7억$
- **포스트-인수 KPI**: 한국 효율화 + 동남아 확장 (베트남·필리핀)
- 의사결정 라인: 가맹사업본부 → CSO → CEO
- 매력: 규모 큼, 동남아 확장 시 Skima도 동반 가능
- 난점: 인수 직후 의사결정 지연 가능, 외국 모회사 보고 라인 복잡

**제안 가치**: 가맹점주 인건비 효율화 → 폐점률 ↓ → 본사 로열티 매출 안정 + 동남아 진출 시 검증된 솔루션

### 9-2. 더벤티
- 매장 수: 1,500+ (빠른 확장 중)
- 본사 조직 작음, 의사결정 빠름
- KPI: 점주 모집 경쟁력 강화
- 매력: **첫 딜 가장 가벼움**, 레퍼런스 케이스 만들기 좋음
- 난점: 규모 컴포즈 대비 작음

**제안 가치**: "더벤티는 점주 운영 편의성도 1등" 마케팅 포인트 — 점주 모집 경쟁력 ↑

### 9-3. 추천 순서

1. **1차 (0~6개월)**: 더벤티 또는 매머드 — **첫 레퍼런스 만들기**
2. **2차 (6~18개월)**: 컴포즈 — 레퍼런스 가지고 진입
3. **3차 (18~24개월)**: 메가커피 — 가장 큰 딜, 마지막

**메가는 처음에 가지 말 것** — 너무 커서 의사결정 6~12개월, 우리 검증 안 된 상태로는 거절. 빽다방은 SPC 본사라 더 어려움.

## 핵심 결론

1. **본사 딜이 사업 성립의 절대 조건** — 24개월 안에 1곳, 36개월 안에 2~3곳. 못 잡으면 사업 불성립.
2. **컴플라이언스 자동화 = 진짜 해자** — 매칭은 따라할 수 있지만 한국 노동법·세무 자동화는 진입장벽 큼.
3. **워커 페르소나의 핵심은 2030 N잡러 + 4050 경력단절 여성** — 학생 중심 가설은 일본 데이터로 반증됨.
4. **노쇼 백업 SLA 정량 조건**: 워커:매장 = 25:1 (등록 기준). 강남 파일럿 = 워커 2,500명.
5. **사업소득자 모델 절대 금지** — 한국 판례·정책 모두 위장도급 방향. 일용근로자 + 점주 직접고용 모델만이 안전.

## Part 11. 단독앱 트랙 (Standalone) — 본사 딜 없이 가는 시나리오

> **전제 변경**: Part 6·9에서 "본사 딜이 절대 조건"이라고 결론냈지만, 그건 *목표 ARR 50억+ / 36개월*을 깔았을 때의 결론이다. 본사 게이트키퍼를 통과 못 해도 사업이 성립하는 길이 있는가? — 이 파트는 단독앱(D2SMB + D2C 워커) 트랙을 처음부터 다시 본다.

### 11-1. 가능성 재평가 — "단독앱은 진짜 안 되나?"

#### (a) 본사 딜 절대 조건론의 반례

| 사례 | 본사 딜 의존도 | 결과 |
|---|---|---|
| **Timee 일본 (2018~2022 초기 4년)** | 거의 0 | 워커 사이드 입소문 + SMB 음식점 직접영업으로 100만 DL 돌파. 대형 체인 딜은 4~5년차부터. |
| **급구 한국** | 낮음 | 누적 200만 DL, Series B(~150억). 카페 수직화·컴플라이언스 약함에도 단독앱으로 도달. |
| **당근알바** | 0 | 하이퍼로컬 단일 채널만으로 자영업 시장 점유. |
| **배민커넥트 초기** | 0 | 라이더 풀 자체 구축 → 나중에 가맹점이 따라옴. |

→ **"본사 딜 없으면 사업 불성립"은 거짓**. 정확한 명제는 *"본사 딜 없으면 50억 ARR 36개월 도달 불가"*이며, **20~30억 ARR 48개월** 트랙은 단독앱으로도 가능.

#### (b) 단독앱이 접근 가능한 매장 풀 재계산

Part 5-2에서 본사 의존 점주 50%를 "비효율"로 분류했지만, 단독앱 관점에서 다시 보면:

| 점주 유형 | 비중 | 단독앱 접근성 | 비고 |
|---|---|---|---|
| 본사 충성 가맹점주 | 50% | △ | 본사 비공식 묵인하 일부 침투 가능. 본사가 자체 솔루션 없으면 제재 약함. |
| **자율형 가맹점주** | 25% | ✅ | 본인 ROI로 결정 — 단독앱 직접 영업 가능 |
| **개인 카페 사장** | 20% | ✅✅ | 본사 지원 0 → 컴플라이언스 자동화가 절대 페인 |
| **베이커리 가맹점주** | 5% | ✅✅ | 새벽 시프트 = 압도적 진입점, 본사보다 점주 의지 |

→ 본사 게이트 우회로 즉시 접근 가능한 풀 = **45% (자율 + 개인) + 5% (베이커리)** = **50%**. 여기에 카페 외 버티컬(편의점·음식점·주점·디저트)을 더하면 사실상 *워커 풀의 동원 한계*가 시장 한계다.

#### (c) 단독앱 LTV/CAC 재계산

Part 6-2에서 직접 영업 CAC 7.5만원을 "본사 딜 ~0원" 대비 비효율로 봤지만, **하이퍼로컬 밀도 + 워커사이드 입소문**으로 끌어내릴 수 있는 마진이 큼.

| CAC 항목 | 기본형 | 밀도형(단일 동 운영) | 비고 |
|---|---|---|---|
| 점주 1매장 (직접영업) | 7.5만원 | **2.5~3만원** | 거점 점주 입소문 + 네이버지도·당근 채널 |
| 워커 1인 (광고+추천) | 1만원 | **0.6만원** | 단일 동 클러스터 형성시 organic 비중 ↑ |

**밀도형 LTV/CAC**:
- 점주: 81만원 / 3만원 = **27x** ✅✅
- 워커: 11.5만원 / 0.6만원 = **19x** ✅✅

→ 본사 딜 100x+에는 못 미치지만, **자본 효율 자체는 단독앱도 충분히 양호**. 핵심은 "단일 동 밀도"를 깨지 않는 것 — 분산 영업하면 CAC 7.5만원으로 회귀.

#### (d) 단독앱 시나리오별 ARR (Part 6-3 대체본)

| 시나리오 | 매장 수 | ARR | 시점 | 누적 워커 |
|---|---|---|---|---|
| 합정·연남 단일 동 시드 | 100 | **0.8억원** | 9개월차 | 2,500명 |
| 마포·서대문 5개 동 | 500 | **4억원** | 18개월차 | 12,000명 |
| 서울 5개 자치구 (마포·강남·서초·송파·용산) | 2,000 | **16억원** | 30개월차 | 50,000명 |
| 수도권 + 부산·대구 광역화 | 5,000 | **40억원** | 48개월차 | 130,000명 |
| 전국 + 본사 inbound 1~2곳 | 8,000~10,000 | **65~80억원** | 60개월차 | 200,000명+ |

→ **단독앱 트랙은 "60개월 ARR 65억" 시나리오가 베이스 케이스**. 본사 트랙 "36개월 50억" 대비 **24개월 느리지만 게이트키퍼 리스크 0**. 본사 딜이 일어나면 부스터일 뿐, 사업의 sine qua non은 아님.

### 11-2. 킥포인트 — 처음 불을 어디에 붙이는가

본사라는 외부 레버가 없을 때, 콜드스타트(닭과 달걀)을 깨는 5개의 킥포인트.

#### 킥 1. **단일 동 + 단일 버티컬 시드** (1순위, 절대 분산 금지)

- **타깃**: 합정·연남·망원 (마포구) — 카페·베이커리·디저트 밀도 1위, 2030 워커 거주 밀도 1위, 인스타·블라인드 organic 노출 수월
- **버티컬**: **베이커리 새벽 시프트(5~7시)** — Part 3 결론과 동일. 업계 최대 페인, 경쟁자 X, 점주 의지 절대적
- **목표 (M0~3)**: 매장 50개 / 워커 200명 manual ops로 시드
- → **밀도가 핵심 KPI**: 매장 1개 반경 500m 안에 워커 50명+ 있을 때만 1시간 매칭 SLA 가능

#### 킥 2. **워커사이드 viral loop** (공급 주도 성장)

Uber식 공급 우선 전략. 일본 Timee의 실제 초기 패턴.

- **즉시 입금 + 추천 보너스 (워커 1인 추천 = 5,000원)** → SNS·블라인드·에브리타임에서 자생
- **워커 등급제 (Verified Barista, Verified Baker)** — 매장 입장에서 "검증된 워커 풀"이 곧 가입 이유
- → 워커 밀도 임계치(동당 2,000명) 도달하면 매장 영업 비용 급락. **공급 → 수요 순서가 직접 영업보다 자본 효율 높음**

#### 킥 3. **점주 챔피언 + 거점 입소문** (Part 5-2 인사이트 재활용)

- 단일 동에서 거점 점주 1~2명을 깊게 만족시키면 → 인근 5~10매장 자동 추천
- 가맹점주 75%가 "본사 또는 주변 점주 영향"으로 결정 — 본사가 없으면 **주변 점주 채널이 유일한 레버**
- → 동별 1~2명의 "에반젤리스트 점주"를 *명시적으로 발굴·관리*. 단순 referral 보너스 X, 운영 파트너로 격상

#### 킥 4. **횡단 워커 (Cross-vertical)** — 워커 LTV 2~3배 증폭

본사 트랙은 단일 브랜드 매장에 워커가 묶이지만, 단독앱은 **같은 워커가 여러 버티컬을 가로지를 수 있음**.

- 5~7시 베이커리 → 9~12시 카페 → 12~14시 점심 음식점 → 같은 워커 1명
- 워커 1인 GMV: 월 4 시프트 → **월 12 시프트**로 3x 증폭 가능
- 워커 LTV: 11.5만원 → **30만원+** (CAC 0.6만원 대비 50x)
- → **매장 입장에서도 검증된 워커 풀이 두꺼워짐**. 단독앱만의 구조적 강점

#### 킥 5. **컴플라이언스 자동화를 매장 후크로** (개인·자율 점주에게 절대 페인)

본사 가맹점주는 본사 표준 양식이 있지만, **개인 카페·자율 점주는 e-근로계약서·일용 원천징수·지급명세서를 본인이 처리해야 함**. 세무사 없이는 거의 불가능.

- "Skima에 매장 등록만 하면 모든 컴플라이언스 자동" → 매장 가입의 *유일하고 충분한 이유*
- 매칭은 보조, 컴플라이언스 SaaS가 진입 후크
- → 매칭 수수료 12% + **컴플라이언스 프리미엄 월 4.9만원** 분리 과금 가능

### 11-3. 전개 방향 — 5단계 로드맵

본사 딜이 없는 전제하의 단계별 전개. 각 단계 완료가 다음 단계의 *전제 조건*.

#### Phase 0 — 시드 (M0~3): 합정·연남 베이커리 새벽 50매장 + 워커 200명

- **운영**: Manual ops, 카톡 채널 + 구글시트 매칭, MVP 앱 최소 기능
- **목표 KPI**: 1시간 매칭률 70%, 매장 NPS 40+
- **검증 가설**: 새벽 베이커리 페인이 진짜인지, 워커 organic 추천이 일어나는지
- **자금**: Pre-seed 5~10억 (창업자 + 엔젤)

#### Phase 1 — 단일 동 풀버티컬 (M3~9): 마포 1동 100매장 + 2,500 워커

- 베이커리 → 카페 → 디저트 → 음식점 → 주점 순으로 버티컬 추가
- **밀도 KPI 달성**: Part 7의 25:1 비율 (매장 100 / 워커 2,500)
- 컴플라이언스 자동화 정식 출시, 프리미엄 과금 시작
- **목표 KPI**: 1시간 매칭률 85%, 월 GMV 3억원, 매장 retention 80%
- **자금**: Seed 15~25억

#### Phase 2 — 인접 5~6동 확장 (M9~18): 마포·서대문 500매장 + 12,000 워커

- 합정에서 검증된 플레이북을 인접 동에 복제 — *동마다 별도 시드 캠페인 X*, 워커 풀이 자연 흘러감
- 점주 챔피언 모델 본격 가동 — 동마다 1~2명 운영 파트너
- **첫 본사 inbound 가능성**: 자율 가맹점주 비중이 높아진 본사(예: 더벤티)가 우리한테 먼저 연락 옴
- **목표 ARR**: 4억원
- **자금**: Series A 50~80억 (단독앱 트랙은 본사 트랙보다 자본 더 필요)

#### Phase 3 — 서울 5개 자치구 (M18~30): 2,000매장 + 50,000 워커

- 마포·강남·서초·송파·용산 등 카페 밀도 상위 자치구 점진 확장
- 이 시점부터 **워커 브랜드 자체가 자산** — "Skima Verified Barista"가 워커 이력서에 적힘
- **본사 협상 레버리지 확보**: 우리가 이미 가맹점주 영향력 행사 중인 상태로 본사 협상
- **목표 ARR**: 16억원
- **자금**: Series B 150~250억

#### Phase 4 — 광역화 + 본사 역방향 영업 (M30~48): 수도권 + 부산·대구

- 더 이상 본사를 우리가 찾아가지 않음 — 본사가 자기 점주 이탈 막으려고 우리한테 옴
- 협상 우위에서 본사 딜 1~2곳 체결 (수수료 인하 대신 *전 매장 일괄 등록*)
- **목표 ARR**: 40억원

#### Phase 5 — 카테고리 정의 (M48~60): 본사 1~2곳 통합 + 자영업 인프라화

- "한국 자영업의 워커·컴플라이언스 OS"로 포지셔닝
- 인접 SaaS 확장 (POS 연동, 급여, 4대보험 통합)
- **목표 ARR**: 65~80억원, IPO/시리즈 D 단계

### 11-4. 본사 트랙과의 비교 요약

| 항목 | 본사 딜 트랙 (Part 9) | 단독앱 트랙 (Part 11) |
|---|---|---|
| 36개월 ARR | 50억원 | 16억원 |
| 60개월 ARR | 100억원+ | 65~80억원 |
| 게이트키퍼 리스크 | **매우 큼** (단일 본사 거절 = 사업 정체) | 없음 |
| 자본 요구량 | 30~60억 (Series A까지) | 50~100억 (Series A까지) |
| 핵심 운영 KPI | 본사 영업 사이클 | 동별 매칭률·밀도 |
| 워커 풀 자산화 | 약함 (브랜드별 격리) | 강함 (횡단 + Verified) |
| Timee 한국 진출 시 방어력 | 본사 딜 = 락인 강함 | 워커 밀도 = 락인 중간 |
| 실패 모드 | 24개월 본사 딜 0 = 폐업 | 60개월 ARR 정체 = 피벗 가능 |

→ **두 트랙은 배타적이지 않음**. 권장 전략은 *Phase 1까지 단독앱으로 검증 → Phase 2 시점에 본사 영업 병행 → Phase 3에서 본사 딜이 들어오면 가속, 안 들어와도 단독으로 완주*. 즉 **본사 딜은 보너스 트랙으로 재정의**하고, 사업 성립의 절대 조건에서 빼는 것이 단독앱 트랙의 본질.

### 11-5. 단독앱 트랙의 진짜 리스크

본사 트랙 리스크(딜 실패)가 사라지는 대신 새로운 리스크가 생김:

1. **자본 효율 악화 → 시리즈 A 어려움**: VC가 "본사 1곳 락인 안 됨 = 해자 약함"으로 평가할 수 있음. → **방어**: 동별 밀도 락인을 정량 지표로 명확히 (1시간 매칭률 90%+, 동별 워커 5,000명+ = supply moat)
2. **Timee 한국 진출 시 정면 충돌**: 자본력으로 밀릴 수 있음. → **방어**: 2027년 전 5개 자치구 락인 + 컴플라이언스 자동화 깊이 (Timee가 1년 안에 못 만드는 수준)
3. **워커 풀 횡단 운영 = 매칭 알고리즘 복잡도 폭증**: 카페·베이커리·음식점·주점 4종을 한 워커가 가로지르면 페어매칭이 N제곱으로 어려워짐. → **방어**: 시드 단계부터 매칭 엔진을 단일 버티컬 X 다중 버티컬 가정으로 설계
4. **개인 카페 점주 디지털 리터러시 한계**: 본사 매장보다 더 낮음. → **방어**: 카카오톡 알림톡 + 웹 대시보드, 앱 강요 X (Part 5-2 시사점 그대로)

## 핵심 결론 (개정)

1. **본사 딜은 절대 조건이 아니라 부스터** — 단독앱 트랙으로 60개월 ARR 65~80억 달성 가능. 본사는 Phase 3 이후 협상 우위에서 진입.
2. **단독앱 트랙의 절대 조건은 "단일 동 밀도"** — 분산 영업 = CAC 7.5만원 회귀, 밀도 운영 = CAC 3만원 + LTV/CAC 27x.
3. **공급(워커) → 수요(매장) 순서가 자본 효율 우위** — Uber·Timee 초기 패턴 재현. 워커 viral loop가 첫 킥포인트.
4. **횡단 워커 모델이 단독앱만의 구조적 강점** — 단일 브랜드 묶이는 본사 트랙에선 불가능. 워커 LTV 3x 증폭.
5. **컴플라이언스 자동화가 개인·자율 점주에게는 본사 트랙보다 더 강한 후크** — 본사가 안 해주는 걸 우리가 해주는 구도.
6. (불변) 사업소득자 모델 절대 금지. 워커:매장 25:1 정량 조건. 일용근로자 모델만 안전.
