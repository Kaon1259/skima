# 스키마 바이트 — Timee 한국 카페 적용 분석

## 1. Timee(일본) 본질 분석

| 요소 | 내용 |
|---|---|
| 창업/현황 | 2017년 창업, 2024년 7월 도쿄증시 상장. 등록 워커 900만+, 등록 사업장 35만+ |
| 핵심 모델 | **면접 X, 이력서 X, 즉시 매칭, 당일 입금** — 1~3시간 단발 시프트도 가능 |
| 신뢰 시스템 | 양방향 별점 평가 (워커 ↔ 점주) |
| 수익구조 | 워커 임금의 **약 30%를 점주에게 수수료** (워커 수수료 0) |
| 법적 구조 | 점주 ↔ 워커 직접 고용 / Timee는 매칭 중개자 (고용주 아님) |
| 성공 배경 | 일본의 극심한 노동력 부족 + 경직된 정규고용 관행 + 모바일/캐시리스 인프라 + F&B의 즉발적 결원 문제 |

핵심은 **"면접·이력서 없이도 작동하는 신뢰 인프라"** + **"당일 페이 UX"** 두 축.

## 2. 한국 시장 컨텍스트

### 카페 시장 (수요측)
- 국내 카페 약 10만 개 이상, 서울 기준 세계 최고 밀도
- **저가 프렌차이즈 폭증**: 메가커피 3,000+, 컴포즈커피 2,600+ (2024 졸리비 인수), 빽다방, 더벤티
- **베이커리 카페**: 파리바게뜨/뚜레쥬르 가맹 + 개인 베이커리
- 마진 구조: 저가 프렌차이즈 GP ~25-30%, 인건비 부담 매우 민감

### 알바 노동시장 (공급측)
- 2025년 최저시급 10,030원 (사상 첫 1만원 돌파)
- 기존 플랫폼: 알바몬/알바천국(이력서 기반), 당근알바(하이퍼로컬), **급구·니더(스팟워크 진입 중)**
- 점주 페인포인트: 주휴수당 회피, 노쇼, 4대보험 가입 부담
- 워커 페인포인트: 근로계약서 미작성, 임금체불, 즉시 페이 부재

## 3. 한국 적용 SWOT

### 🟢 기회 (Opportunities)
1. **인구구조 변화로 일본화 진행 중** — 카페 알바 구인난 가속
2. **카페 수요 밀집도 세계 최고** — TAM 충분
3. **Toss/카카오페이 인프라**로 당일 정산 기술적으로 가능
4. **순수 스팟워크 시장은 아직 미정착** — 급구/니더가 있지만 카페 수직화 X
5. **저가 프렌차이즈 본사 단위 딜** 가능 (5,000+ 매장 일괄 온보딩)

### 🔴 리스크 (Risks)
1. **근로기준법 마찰** — 1일 알바도 근로계약서 의무, 산재 가입, 일용근로자 세무처리(3.3% vs 근로소득) 복잡
2. **수수료 천장 낮음** — 일본 30% 불가능. 한국 카페 마진상 **10~15%가 현실적 상한**
3. **기존 경쟁자**: 급구(Series B), 니더, 당근알바 하이퍼로컬, 알바콜
4. **프렌차이즈 본사 통제** — 점주가 외부 플랫폼 자유롭게 못 쓸 수 있음 → 본사 영업 필수
5. **문화적 장벽** — 한국 알바는 여전히 관계 기반, 익명·평점 기반 채용 저항

## 4. 차별화 전략 (vs. 급구/니더/당근알바)

| 차별화 축 | 구체 액션 |
|---|---|
| **카페 수직화** | 메가/컴포즈/빽다방 POS 사전교육 모듈, "카페 인증 워커" 등급 |
| **B2B2C — 본사 딜 우선** | 점주 한 명씩 영업 X, 컴포즈/빽다방 본사 한 건이 5,000매장 |
| **컴플라이언스 인프라화** | 근로계약서 자동 e-sign, 3.3% 원천징수 자동, 산재 자동신고 — **소형 카페가 절대 혼자 못함, 진짜 해자** |
| **당일 페이 (Toss 연동)** | 급구 T+N → 차별점 |
| **베이커리 새벽 시프트** | 5~7시 빵 굽기/진열 — 가장 안 채워지는 시프트, 단가↑ → 수수료율↑ 정당화 |
| **워커 스킬 배지** | 에스프레소/라떼아트/POS/베이킹 — Verified Barista 티어 |

## 5. 가장 먼저 결정해야 할 것 (의사결정 포인트)

1. **고용 구조**: 점주 직접고용(Timee식) vs. 사업소득자(3.3%) vs. 일용근로자 — **법무·세무 검토 1순위**, 모든 기능이 여기 종속됨
2. **GTM 방향**: 프렌차이즈 본사 우선(스케일 크지만 영업 사이클 6~12개월) vs. 인디 카페 우선(빠르지만 단가 낮음)
3. **지리 전략**: 강남/홍대 1개 동 점령(밀도 효과) vs. 서울 전역 분산
4. **포지셔닝**: 저가(수수료 5~8%) vs. 프리미엄(수수료 12~15% + 컴플라이언스/검증)

## 6. MVP 권장 범위

**Phase 1 (3~6개월) — 1개 상권 검증**
- 타겟: 강남 또는 홍대, 카페 100개(프렌차이즈 60 + 인디 30 + 베이커리 10)
- 워커: 카페 경험자 1,000~2,000명
- 핵심 플로우: 시프트 등록 → 1탭 지원 → 근무 → 양방향 평가 → T+1 정산 (이후 당일)
- 컴플라이언스: 일용근로자 3.3% 원천징수 + 근로계약서 e-sign 자동화

**핵심 KPI**
- Fill Rate (1시간 내 시프트 충원율) — Timee 벤치 80%+
- 노쇼율 — 목표 <3%
- 재고용률 (점주가 같은 워커 재예약) — PMF 가장 강한 시그널
