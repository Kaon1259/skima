# Skima Byte의 킥(Kick) — 단 하나의 결정적 약속

## 결론

> **킥 = "1시간 매칭, 30분 입금"**
>
> - 점주: 시프트 등록 후 **1시간 내 매칭 보장** (실패 시 다음 시프트 수수료 무료)
> - 워커: 근무 종료 후 **30분 내 입금 보장** (Toss 오픈뱅킹 연동)

이 두 줄이 서비스의 모든 것을 정의한다. 이 두 줄을 못 지키면 또 하나의 알바 앱일 뿐이다.

---

## 왜 "시간"이 킥인가

### 1. 한국 시장은 "시간 SLA의 정량화"에 학습 완료
- 배민 "30분 배송" → 음식 배달 카테고리 정의
- 쿠팡 "로켓배송" → 이커머스 재정의
- 토스 "초간편 송금" → 간편결제 정의

한국에서 카테고리를 만드는 공식 = **시간 약속을 카테고리 이름으로 박는 것**. 검증된 패턴.

### 2. 경쟁자가 약속 못 한 단 하나

| 플랫폼 | 매칭 시간 | 입금 시간 |
|---|---|---|
| 알바몬 | 며칠~주 (면접 후) | 월급 |
| 당근알바 | 수 시간~일 (메시지 협상) | 협의 |
| 급구 | 가능하나 SLA 미보장 | T+N |
| **Skima Byte** | **1시간 보장** | **30분 보장** |

이걸 못 한 건 "안 한 것"이 아니라 "못한 것". 워커 풀(25:1 비율) + 매칭 엔진 + 오픈뱅킹 자동 정산 인프라 없이는 약속 자체가 불가능. 이게 진입장벽.

### 3. 양쪽 시장을 한 슬로건으로 후크
- 점주 페인 1위(노쇼/펑크) 직격 = "1시간 매칭"
- 워커 페인 1위(돈 빨리 받고 싶음) 직격 = "30분 입금"
- 한 슬로건이 양면 시장 모두를 잡음 = 카테고리 마케팅 효율 압도적

### 4. 한국이 일본 Timee보다 더 강하게 갈 수 있는 영역
- Timee 일본 "当日払い" = 실제론 일 단위 배치 처리, "30분 내" 약속 없음
- 한국 Toss/오픈뱅킹 인프라 = 실시간 송금 표준
- 일본 모방이 아니라 한국 인프라로 한 단계 위 약속

---

## 킥을 지키는 "보이지 않는 인프라"

고객한테는 시간만 팔고, 인프라는 숨긴다.

| 보이는 약속 (킥) | 숨겨진 인프라 (해자) |
|---|---|
| 1시간 매칭 | 워커 풀 25:1 비율, 매칭 엔진, 푸시 즉시성 |
| 30분 입금 | Toss 오픈뱅킹, 일용근로 원천징수 자동, 정산 회계 |
| (암묵) "그냥 알아서 됨" | e-근로계약서, 산재 자동 신고, 4대보험 알림 |

**핵심 원칙**: 점주는 컴플라이언스 가치를 미리 모름. 사후에 깨닫고 이탈 못 함. 그래서 컴플라이언스는 **킥이 아니라 락인(lock-in)**.

## 킥의 진화 — 단계별 약속 갱신

| Phase | 기간 | 킥 | 의미 |
|---|---|---|---|
| 1 | 0~12개월 | "1시간 매칭, 30분 입금" | 카테고리 정의 |
| 2 | 12~24개월 | **"노쇼 0% 보장"** (자동 백업 + 위약금) | 점주 결정적 심리 안전 |
| 3 | 24~36개월 | **"카페 운영 OS"** (매칭+컴플라이언스+인력 데이터) | B2B SaaS 진화, 본사 단위 표준화 |

킥은 진화한다. 처음부터 다 약속하면 다 못 지킴.

## 안티 킥 — 가지 말아야 할 약속

| 안티 킥 | 왜 안 되는가 |
|---|---|
| "최저 수수료" | 가격 경쟁 = 카테고리 정의 안 됨, 마진 침해 |
| "최대 워커 풀" | 자랑일 뿐, 고객 후크 X |
| "AI 매칭" | 모두가 함, 차별 X |
| "프리미엄 워커만" | 풀 작아 매칭 실패 → 킥 깨짐 |
| "전국 어디서나" | 밀도 분산 → 1시간 약속 못 지킴 (밀도가 킥의 전제) |

## 킥이 깨지는 시나리오 — 양방향 붕괴

```
1시간 매칭 깨짐 → 점주 이탈 → 워커 일감 ↓ → 워커 이탈
                                              ↓
30분 입금 깨짐 → 워커 이탈 → 매칭 풀 ↓ → 1시간 약속 깨짐
```

두 약속은 **상호 의존**. 어느 한쪽 깨지면 양방향 붕괴.

**운영 북극성 KPI — 단 2개**:
1. 매일 시프트의 1시간 내 매칭률 (목표 80%+)
2. 매일 정산의 30분 내 입금률 (목표 95%+)

대시보드 1면에 이 두 숫자만 박는다. 나머지는 보조 지표.

- "급할수록 스키마"

- "8시 노쇼? 9시엔 다른 알바가 옵니다"
- "오늘 펑크나도 오늘 바로 채웁니다"

### 워커 광고 메시지
- "퇴근 30분 후 통장에 입금"
- "면접 없이 1탭 응시"

### 본사 영업 메시지
- "가맹점주 폐점 사유 1위 = 인건비 = 우리가 1시간 안에 해결"
