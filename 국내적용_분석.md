# 국내 적용 — 중심 요소 + 한국 시장 현황 분석

## Part 1. 한국 적용 시 중심에 둘 4가지 축

### 축 1. 카페 페인포인트의 우선순위 (점주 측)

한국 카페 점주의 진짜 1순위 페인은 **"갑작스런 노쇼/펑크"**. 알바몬·당근알바도 채용은 되지만, 펑크가 났을 때 **3시간 안에 대체 인력 투입**이 안 됨. 여기서 시작하는 게 정답.

| 우선순위 | 페인 | 현재 시장 해결 여부 |
|---|---|---|
| 1 | 당일 노쇼/펑크 → 점주가 들어가야 함 | ❌ 미해결 (가장 큰 빈자리) |
| 2 | 채용 면접·교육 시간 비용 | △ 부분 해결 (Timee식 X) |
| 3 | 주말/이벤트 변동성 | △ 부분 해결 |
| 4 | 근로계약서·세무 컴플라이언스 | ❌ 미해결 (B2B SaaS 따로 있음) |

→ **노쇼 백업 워커 풀 + 1시간 내 매칭 SLA**가 진입 무기.

### 축 2. 워커(알바) 결정 후크

20대 알바가 Timee로 이동한 결정적 이유 = **당일 입금**. 한국도 동일 적용:

- Toss/카카오페이 인프라가 일본보다 우수 → 기술적 장벽 X
- 학생·N잡러 현금흐름 압박 → "근무 종료 30분 후 입금"이 압도적 후크
- 면접 부담 제거 + 평점으로 악성 점주 거르기

### 축 3. 한국 노동법 — 어떤 분류로 갈 것인가 (가장 큰 결정)

| 분류 | 적합성 | 리스크 |
|---|---|---|
| **일용근로자** (1일 단위) | ★★★ Timee식 1일 매칭에 가장 fit | 같은 매장 월 8일 초과 시 4대보험 의무 발생 |
| **단시간근로자** | ▲ 정기 반복 시프트에 적합 | 주휴수당 계산 복잡 |
| **사업소득자(3.3%)** | ✕ 비추천 | 카페 알바는 종속노동 → **위장도급 적발 시 형사·과태료** |

**권장**: 기본은 일용근로자 분류 + 원천징수(일용근로 6.6%) 자동 + 같은 매장 누적 알림(자동 4대보험 트리거). 절대 사업소득자(3.3%) 모델로 가지 말 것 — 라이더유니온/플랫폼노조 리스크.

### 축 4. 차별화 — 한국 경쟁자 대비 4개 빈자리

급구·니더·당근알바 누구도 채우지 못한 4가지:

1. **카페 수직화** (POS 사전교육, 바리스타 등급제) — 누구도 안 함
2. **컴플라이언스 자동화** (e-근로계약서 + 원천징수 + 지급명세서) — 다 약함
3. **프렌차이즈 본사 B2B2C 딜** — 누구도 본사부터 안 침
4. **워커 스킬 인증** (Verified Barista) — 시장 전체에 부재

이 4개 중 **1개라도 깊게** 하면 포지셔닝 확보, **3개 이상**이면 카테고리 정의 가능.

## Part 2. 현재 국내 상황 (2025 기준)

### A. 카페 산업

- 전국 커피 전문점 **약 10만 개** (통계청), 서울 자치구 평균 1,000+
- **저가 프렌차이즈 포화기**: 메가 3,400+ / 컴포즈 2,600+ (2024 졸리비 인수, 4.7억$) / 빽다방 1,500+ / 더벤티·매머드 급성장
- 3년 생존율 50% 이하, 폐업 사유 1위 = **인건비**
- 매장당 인건비 비중: 매출의 **25~35%** → 1%p 효율 개선이 곧 생존

### B. 알바·스팟워크 시장 지형

| 플레이어 | 포지션 | 한계 |
|---|---|---|
| 알바몬 (잡코리아) | 이력서 기반 1위, MAU 800만+ | 1일 매칭 부적합 |
| 알바천국 | 이력서 기반 2위 | 알바몬과 동일 한계 |
| 당근알바 | 하이퍼로컬, 빠른 성장 | 매칭 알고리즘·즉시성 약함 |
| **급구** | 2018년 창업, Series B(~150억), 누적 200만 DL | **카페 수직화 X, 컴플라이언스 약함** |
| 니더 | 2022년 창업, 단기 매칭 | 규모 작음 |
| Timee 본체 | 아직 한국 미진출 | **2027 전 골든 윈도우** |

### C. 거시 환경 — "일본화" 가속

1. **초고령사회 진입** (2025년 65세+ 비율 20% 돌파) → 청년 알바 풀 축소 = 일본 2010년대 재현
2. **N잡 인구 60만 돌파** (2024) → 단기/유연 근무 정상화
3. **금융 인프라 세계 최고**: Toss MAU 1,500만+, Kakao Pay 2,400만+ → 당일 정산 인프라 ✅
4. **정부 정책**: 플랫폼노동자 보호법 진행 중 → **선제 컴플라이언스 = 정책 우호 + 해자**

### D. 인건비 충격

- 2025년 최저시급 **10,030원** (사상 첫 1만원 돌파, +1.7%)
- 주휴수당 포함 실질 시급 ~12,036원
- 점주는 **15시간 미만으로 시프트 쪼개기** → Skima Byte 모델과 자연스럽게 fit

## Part 3. 결론 — 집중해야 할 3가지

### 제품 3대 무기
1. **1시간 내 노쇼 백업 매칭** (점주 페인 1위 직격)
2. **자동 e-근로계약서 + 일용 원천징수 + 지급명세서** (점주 절대 혼자 못함 = 진짜 해자)
3. **카페 인증 워커 풀** (POS·에스프레소·라떼아트 사전 검증)

### 시장 3대 거점
1. **저가 프렌차이즈 본사 1곳 파일럿** — **컴포즈** 또는 **더벤티** 추천 (메가는 너무 큼, 빽다방은 SPC 본사 어려움)
2. **강남 또는 홍대 1개 동 밀도 우선** — 워커 회전율이 핵심
3. **베이커리 새벽 시프트(5~7시)** = 진입점 — 업계 최대 구인난 구역

### 타이밍
- **2025~2027 = 골든 윈도우**: Timee 한국 진출 전, 일본화 본격화 직전
- 2028년 이후 진입은 늦음

## Part 4. 구체적 다음 액션 후보

1. **컴포즈/더벤티 본사 IR 자료** — 본사 입장 ROI 제시 (점주 이탈률 감소, 매장 운영 표준화)
2. **노쇼 백업 SLA 모델링** — 1시간 내 매칭 보장하려면 매장 1개당 워커 풀 몇 명 필요한가
3. **컴플라이언스 자동화 MVP 스펙** — e-근로계약서, 일용 원천징수, 지급명세서 자동 제출 흐름
4. **법무 자문 1차 검토** — 일용근로자 분류 / 플랫폼 책임 범위 / 산재 가입 의무 명확화
